Examples of My Work

I'm currently working on moving a lot of my personal projects to my GitHub account!

ENSE

An application written for the buying consortium ENSE. It is built on Node.js and RethinkDB. It features a member section where users can find information about other companies in the consortium. Code snippets can be furnished upon request.

Insta-Average

An application that takes 20 popular images from the Instagram API every hour and calculates the average color of the image. It then stores it in RethinkDB in order to run queries over to find the average color of photos posted to Instagram.

Requirements Indexer

An Eclipse extension written as a team of four for our Software Engineering Lab. It reads .txt files that contain project requirements, and indexes them. It also can index a project's source code when the application is loaded.

Image Manipulator

An iPhone app written for the Intro to iPhone Development class. We created 5 different transformations for a given image. The transformations were applied on a pixel-by-pixel basis and the resulting image could be saved to the photo library.
